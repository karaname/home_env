## Usage
```
mc         creates useful configurations for midnight commander (from configs/mc)
vim        sets up a text editor / sets .vimrc (from configs/etc) and install some plugins I like
home       removes unnecessary directories from $HOME
bashrc     adds a few lines to bashrc (exports / alias)
hotkeys    sets up hotkyes which I use often on my Manjaro Xfce
packages   installing packages that often come in handy for me (pacman)
```
